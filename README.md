## ✨ Features

### 📄 DocChat
- Upload textbooks, PDFs, or notes.
- Ask questions in natural language.
- Context-aware answers from your own study materials.

### 📝 Smart Notes
- Organize revision material.
- Track LeetCode questions and coding progress.
- Add personal notes and build dynamic study plans.

### 🤖 Personal AI Assistant
- Solve doubts instantly.
- Summarize lengthy content.
- Adaptive and personalized support for your learning path.

### 🧠 Flashcards Maker
- Automatically generate flashcards from any content.
- Practice with **spaced repetition** for long-term memory.

### 🎯 And Much More...
- 📅 Personalized study plans  
- 🎯 Goal setting and performance tracking  
- 🧠 AI-powered content generation  
- 🔗 Resource and topic recommendations
